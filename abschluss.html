Retrospektive Abschluss

Retro bewerten

Die Vergabe von Schulnoten (1= sehr gut, 6=sehr schlecht) von den Teilnehmern ist eine Möglichkeit für die Bewertung einer Retrospektive. Dies geschieht anonym.

Material: Post it

Dauer: 1 Minute

Token of Gratitude

Am Ende der Retro wird von jedem Teammitglied an jeweils eine Person ein Token of Gratitude vergeben. Dieser wird verteilt an diejenigen, welche man während des letzten Sprint besonders geschätzt hat. Den Token kann man z. B. vergeben, wenn man von einer anderen Person besonders viel Unterstützung bekommen hat, jemand herausragende Arbeit geleistet hat oder eine gute Lösung für ein Problem im Team entwickelt hat. Dieser Token wird dann übergeben (man kann z. B. kleine Medaillen nutzen) und es wird vor dem Team gesagt, warum man diesen Token übergibt.

Material: Tokens

Dauer: 5-10 Minuten
